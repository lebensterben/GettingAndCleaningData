# Code Book

There should be 68 variables, where the first two are identifiers for each individual test subject and for the activity the test sbuject was performing during the sampling window.

The remaining 66 variables are measurements collected by the accelerator and gyroscope of the Samsung phone.

In the variable names, a `t` represents time domain variables while a `f` represents frequency domain variables, which is obtained via fast fourier transformation.

More details could be found in "UCI HAR Dataset\README.txt" and "UCI HAR Dataset\features_info.txt".
